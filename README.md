# MODERN-TO-DO-TASK-MANAGER
A simple and responsive **Todo Task Manager** built using **HTML, CSS, and JavaScript**. It helps users organize their daily tasks by adding, editing, completing, and deleting tasks.

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 💾 Saves tasks using Local Storage
- 📱 Responsive and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
